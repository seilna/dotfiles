# dotfiles
---
## Installation
```
$ curl -fsSLo ~/.zshrc https://raw.githubusercontent.com/seilna/dotfiles/master/.zshrc && \
curl -fsSLo ~/.tmux.conf https://raw.githubusercontent.com/seilna/dotfiles/master/.tmux.conf && \
curl -fsSLo ~/.vimrc https://raw.githubusercontent.com/seilna/dotfiles/master/.vimrc && \
source ~/.zshrc ~/.tmux.conf ~/.vimrc
```
